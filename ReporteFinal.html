Solucion al problema de clasificacion de tipos de viaje a Walmart

Introduccion

Whether theyre on a last minute run for new puppy supplies or leisurely making their way through a weekly grocery list, classifying trip types enables Walmart to create the best shopping experience for every customer. Currently, Walmarts trip types are created from a combination of existing customer insights (art) and purchase history data (science). In their third recruiting competition, Walmart is challenging Kagglers to focus on the (data) science and classify customer trips using only a transactional dataset of the items theyve purchased. Improving the science behind trip type classification will help Walmart refine their segmentation process.

Modelado - Apliquemos machine learning

El objetivo de este proyecto era aplicar 3 algortmos de aprendizaje de mquina que nos ayudaran a clasificar el dataset de pruebas y medir nuestro score en Kaggle

Algoritmo 1 -> KNN

El primer algortmo que aplicamos fue el KNN utilizando 5 vecinos cercanos, y una votacin mnima = 0.

Al aplicar este algoritmo tuvimos que aplicar una columna de rudo al dataset, ya que al intentar predecir la clasificacind el set de Pruebas notamos que haba muchos empates en la clasificacin de salida.

Algoritmo 2 -> XGBoost

Aplicamos el XGBoost con los siguientes parmetros: * multi = softprob * eval_metric = mlogloss * num_class = 37

Se utiliz cross validation para el entrenamiento del modelo con los siguientes parmetros:

  • Round = 50
  • Folds = 5

import pandas as pd
import numpy as np
import scipy as sp
from sklearn.linear_model import LogisticRegression
import time

start_time = time.time()

waltrain = pd.read_csv('C:/Users/lucky/Documents/Ciencia de Datos/Aprendizaje de Maquina/Proyecto Final ML/train/train.csv')
waltest = pd.read_csv('C:/Users/lucky/Documents/Ciencia de Datos/Aprendizaje de Maquina/Proyecto Final ML/test/test.csv')
waltrain = waltrain[waltrain.FinelineNumber.notnull()]
waltrain_part = waltrain[:]
waltest_part = waltest[:]

model = LogisticRegression()
x = waltrain_part[['Weekday', 'DepartmentDescription']]
y = waltrain_part[['TripType']]
x = pd.get_dummies(x)
z = waltest_part[['Weekday', 'DepartmentDescription']]
zend = pd.DataFrame({'Weekday': ['Sunday'],
'DepartmentDescription': ['HEALTH AND BEAUTY AIDS']},
index = [len(z)])
z = z.append(zend)
z = pd.get_dummies(z)

model.fit(x, y)
print ("The model coefficients are:")
print (model.coef_)
print ("The intercepts are:")
print (model.intercept_)
print ("model created after %f seconds" % (time.time() - start_time))

submission = model.predict_proba(z)
submissiondf = pd.DataFrame(submission)

submissiondf.drop(len(submissiondf)-1)

dex = waltest.iloc[:,0]
